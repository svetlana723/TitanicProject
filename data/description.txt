ДАННЫЕ ТИТАНИК — ОПИСАНИЕ ПОЛЕЙ

PassengerId — уникальный номер пассажира. Просто идентификатор, никакой информации не несёт.

Survived — выжил или нет. Это наша целевая переменная (то, что будем предсказывать).

    0 = погиб

    1 = выжил

Pclass — класс билета (социальный статус).

    1 = первый класс (богатые)

    2 = второй класс (средний класс)

    3 = третий класс (бедные)

Name — имя пассажира. Содержит фамилию, имя и иногда титул (Mr, Mrs, Miss, Master и др.).

Sex — пол.

    male = мужчина

    female = женщина

Age — возраст в годах. Много пропусков.

SibSp — количество братьев, сестёр или супругов на борту.

    отдельно муж/жена считаются

    отдельно братья/сёстры считаются

    0 = путешествовал один (без этих родственников)

Parch — количество родителей или детей на борту.

    родители (мать, отец)

    дети (дочь, сын)

    0 = путешествовал один (без этих родственников)

Ticket — номер билета. Может быть одинаковым у членов семьи.

Fare — цена билета в английских фунтах.

Cabin — номер каюты. Очень много пропусков (у кого не было каюты).

    первая буква номера — палуба (A, B, C, D, E, F, G)

Embarked — порт посадки.

    S = Саутгемптон (Англия) — начальная точка

    C = Шербур (Франция) — первая остановка

    Q = Квинстаун (Ирландия) — последняя остановка перед отплытием в Америку
